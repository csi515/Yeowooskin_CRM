# 아이콘 파일

PWA 아이콘 파일을 이 디렉토리에 배치하세요.

## 필수 파일

- `icon-512.png`: 512x512 픽셀 PNG 형식

## 권장 파일

- `icon-192.png`: 192x192 픽셀 PNG 형식
- `icon-256.png`: 256x256 픽셀 PNG 형식
- `icon-384.png`: 384x384 픽셀 PNG 형식
- `icon-512.png`: 512x512 픽셀 PNG 형식

## 생성 방법

1. 원본 이미지를 준비합니다 (최소 512x512 권장)
2. 이미지 편집 도구(Photoshop, GIMP, 온라인 도구 등)를 사용하여 각 크기로 리사이즈합니다
3. PNG 형식으로 저장합니다
4. 이 디렉토리에 배치합니다
